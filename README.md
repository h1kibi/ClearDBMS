# ClearDBMS

基于 RUCBase 实验项目开发的教学型关系数据库管理系统，支持 SQL 解析、执行、索引、事务与并发控制等核心功能。

## 实验环境

- 操作系统：Ubuntu 18.04 及以上 (64位)
- 编译器：GCC
- 编程语言：C++17
- 管理工具：cmake
- 推荐编辑器：VSCode

### 依赖环境库配置

- gcc 7.1 及以上版本（要求完全支持 C++17）
- cmake 3.16 及以上版本
- flex
- bison
- readline

## 功能特性

- SQL 解析：基于 lex/yacc 的 SQL 语句解析
- 查询执行：支持投影、选择、连接、排序、聚合等算子
- 索引管理：B+ 树索引实现
- 事务管理：ACID 事务支持
- 并发控制：基于锁的并发控制机制
- 日志恢复：WAL 日志与故障恢复
